# Развертывание на Vercel

## Шаг 1: Регистрация
1. Перейди на https://vercel.com
2. Нажми "Sign Up"
3. Выбери "Continue with Google"
4. Войди под почтой senseirabochaya@gmail.com

## Шаг 2: Импорт проекта
1. После входа нажми "New Project"
2. Выбери "Import Git Repository"
3. Найди репозиторий "JIAMA4455/creator-hub-ui"
4. Нажми "Import"

## Шаг 3: Настройка
1. Оставь все настройки по умолчанию
2. Нажми "Deploy"

## Шаг 4: Готово!
1. Через 1-2 минуты получишь URL вида: `creator-hub-ui.vercel.app`
2. Этот URL будет работать лучше чем GitHub Pages
